## What it does

- **59,000+ Oracle Fusion endpoints** across HCM, FSCM, and BPM — fully indexed and searchable offline
- **Visual Q Builder** — construct `q` query filters field-by-field without memorizing syntax
- **Finder Builder** — pick and configure Finders for any endpoint visually
- **Schema viewer** — browse full request and response models for every endpoint
- **Live requests** *(Pro)* — send requests to your Oracle instance and inspect real JSON responses
- **Flow runner** *(Pro)* — chain multiple API calls into reusable, shareable automations
- **Export & backup** *(Pro)* — portable export of flows, environments, and history

Works completely offline. No Oracle instance required to explore the catalog.
