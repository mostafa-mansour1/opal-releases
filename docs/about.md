# About

OPAL is built by **Mostafa Mansour**, a Technical Lead and Senior Software Engineer.

OPAL is a solo project. I built it because I spent years working with Oracle Fusion Cloud APIs and got tired of digging through hundreds of pages of documentation just to find the right endpoint or figure out the correct Q filter syntax. OPAL is the tool I wished existed.

Every feature came from a real frustration — the visual Q Builder, the Finder Builder, the offline-first catalog — all things that would have saved me hours on real projects.
